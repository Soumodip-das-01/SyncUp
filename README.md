# SyncUp

A modern social media platform with real-time chat built using Next.js, Express.js, MongoDB, Socket.IO, and Tailwind CSS.

## Features

- Authentication
- Social Feed
- Real-time Chat
- User Profiles
- Stories
- Notifications
- Follow/Unfollow
- Responsive UI

## Tech Stack

- Next.js
- Express.js
- MongoDB
- Socket.IO
- JWT
- Tailwind CSS
